# vue-bolg
### 执行一下操作开启项目
<br/> git clone 项目地址
<br/> cd client npm i
<br/> cd server npm i
<br/>cd shujuku
 <br/>npm i sequelize-cli
<br/> cd shujuku
<br/> npx sequelize db:create
<br/> npx sequelize db:migrate
